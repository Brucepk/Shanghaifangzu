大家还记得你们刚来上海的时候租房的房租是多少吗？现在相比于那个时候，是不是涨了不少？上个月中介哄抢租赁房源的事件大家应该还没忘记。pk哥在上海，也是见证了房租稳步上涨的过程。很多网友感慨，以前是买不起房，现在是租房都租不起了。

今天，pk哥用 Python 爬虫给大家分析下上海的房租。我们用数据来看看上海的房租究竟有多高。

这次的租房房源数据来源于链家网，用 Python 收集数据后制作成可视化图形。


#### 效果图
